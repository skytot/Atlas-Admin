# 安全与运维

涵盖登录安全、权限审计、运维监控、告警体系等，确保大型管理平台的可靠性与合规性。

## 安全能力

- 登录保护：双因子认证、登录告警、会话管理
- 数据安全：敏感字段脱敏、权限审计、操作日志
- 安全扫描：CI 中集成依赖漏洞扫描、静态代码扫描

## 运维能力

- 监控：接入 Sentry/TrackJS、性能指标采集
- 告警：异常阈值配置、通知策略、值班流程
- 配置中心：动态配置项管理、公告、版本信息
